# Font Candidates for Mouse House

To preview all fonts visually in-app: **Right-click pet → Appearance → Preview Fonts**

The preview panel shows each font at 22px with a sample sentence. Scroll to compare all 15 fonts.

---

## Pixel / Retro Fonts

| Font | Style | Best For |
|------|-------|----------|
| **Press Start 2P** | Chunky 8-bit arcade | Titles, retro game UI |
| **Silkscreen** | Clean minimal pixel | Menus, UI labels, body text |
| **Silkscreen Bold** | Bold variant | Emphasis, headers |
| **VT323** | Terminal / VT320 | Console text, status displays |
| **DotGothic16** | Japanese dot-matrix | Body text, multilingual (CJK) |
| **Pixelify Sans** | Modern rounded pixel | Friendly UI, titles |
| **Tiny5** | Ultra-compact 5px | Tiny labels, pixel art captions |

## Display / Decorative Fonts

| Font | Style | Best For |
|------|-------|----------|
| **Jersey 10** | Condensed sport numerals | Scores, compact displays |
| **Jersey 15** | Wider sport variant | Titles, larger displays |
| **Bungee Shade** | 3D block letters | Big splash titles |
| **Jacquard 12** | Ornamental cross-stitch | Decorative headers |
| **Matemasie** | Playful rounded display | Fun titles, kid-friendly |

## Tech / Clean Fonts

| Font | Style | Best For |
|------|-------|----------|
| **Share Tech Mono** | Clean monospace | Code, coordinates, status |
| **Geo** | Geometric sans | Clean modern UI |
| **Orbitron** | Futuristic geometric | Sci-fi UI, tech displays |

## Recommendations for Mouse House

| Use Case | Top Pick | Runner-up |
|----------|----------|-----------|
| **Menu text / UI** | Silkscreen | VT323 |
| **Activity titles** | Pixelify Sans | Press Start 2P |
| **Body text** | Silkscreen | DotGothic16 |
| **Scores / numbers** | Jersey 10 | Press Start 2P |
| **Small labels** | Tiny5 | Silkscreen |
| **Status bubble** | Silkscreen | VT323 |
